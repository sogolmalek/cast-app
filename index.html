Cast eliminates every onboarding step between an AI agent and Nosana's decentralized GPU grid. No API keys, no credit purchases — just HTTP + micropayment + compute.

⛔ Without Cast
  • 1
    Create Account — Sign up, verify email
    ↳ Human step. Agent can't do this.
  • 2
    Generate API Key — Dashboard → create key
    ↳ Key management per instance.
  • 3
    Purchase Credits — NOS tokens or card
    ↳ Requires exchange account.
  • 4
    Create Deployment — Job def, GPU market, replicas
    ↳ Config overhead per model.
  • 5
    Monitor Health — Poll status, handle failures
    ↳ Agent implements failover.
  • 6
    Manage Billing — Track balance, top up
    ↳ Human intervention loop.
6 steps · 3 human interventions · ~30 min
